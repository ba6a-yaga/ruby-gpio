# Ruby-GPIO

A Linux sysfs-based GPIO interface wrapper

## Installation

Add this line to your application's Gemfile:

    gem 'ruby-gpio'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruby-gpio
